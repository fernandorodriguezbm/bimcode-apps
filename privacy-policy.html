SectionMove Pro — Privacy Policy

Publisher: BIMCode (Luis Fernando Rodriguez Coarite)  |  Effective date: June 10, 2026

1. Data the application collects and how it is used

SectionMove Pro is an add-in for Autodesk® Revit®. It does not collect, store, or transmit any personal information, project data, model data, files, or usage analytics.

The only data the add-in processes is your Autodesk account user ID (the identifier provided by Revit as LoginUserId). This ID is sent exclusively to Autodesk's official Entitlement API (apps.exchange.autodesk.com) for one purpose only: to verify that your Autodesk account holds an active subscription to SectionMove Pro purchased through the Autodesk App Store.

The result of this verification (valid / not valid, plus a timestamp) is cached locally on your computer, encrypted with Windows Data Protection API (DPAPI), so the add-in can keep working for up to 7 days without an internet connection. This cache never leaves your computer.

2. Third parties

The add-in shares data with only one third party: Autodesk, Inc., which operates the App Store and the Entitlement API used to validate your subscription. Autodesk processes this data under its own privacy statement ( https://www.autodesk.com/company/legal-notices-trademarks/privacy-statement), which provides at least the same level of protection described in this policy.

Payment processing for purchases is handled entirely by Autodesk and its payment providers; the publisher never receives or stores your payment details. The add-in contains no analytics tools, advertising networks, or third-party SDKs that collect data.

3. Data retention and deletion

The publisher does not operate any server and therefore retains no user data. The only stored item is the local, encrypted license cache on your own computer, located at %AppData%\SectionMovePro. It is overwritten on each successful verification and you can delete it at any time by removing that folder; it is also removed when you uninstall the application.

4. Your rights — revoking consent and data deletion

Because the add-in stores no personal data on the publisher's side, no deletion request to the publisher is necessary: uninstalling the add-in and deleting the folder named above removes all locally stored data. To stop the subscription verification entirely, uninstall the add-in or cancel your subscription in your Autodesk Account. For data held by Autodesk (account, purchases), please refer to Autodesk's privacy statement linked above.

For any privacy question or request, contact the publisher at fernando.rodriguez.bm@gmail.com. We reply within 2 business days.

Changes to this policy

If the application ever changes the data it processes, this policy will be updated at this same URL and the effective date revised.
